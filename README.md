## Features

- Dynamic Number Animation

    Animated counter for the cycle count and year display using Intersection Observer API.

- Smooth Navigation

    Fixed navigation bar for convenience.

- Responsive Design

    Mobile and window friendly layout that adapts to different screen sizes

- Contact Form

    Fully functional form with email validation (It is just there for the looks, that is not sending out any message to me)

- Social Links

    Included social media via clickable logos for easy access like YouTube, Instagram, GitHub

- Video Integration

    Integrated a video for a trailer that users can know more about the topic more easily
